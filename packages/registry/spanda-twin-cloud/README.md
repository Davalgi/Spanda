# spanda-twin-cloud

Official Spanda package: **Twin Cloud SaaS sync**

## Import

```spanda
import twin.cloud;
```

## Capabilities

Requires `cloud.invoke` for remote twin snapshot upload via the Twin Cloud HTTP API.

## Status

Pairs with `spanda twin cloud` CLI and Control Center `/v1/twins/*` REST backend.

See [docs/twin-cloud.md](../../../docs/twin-cloud.md).
