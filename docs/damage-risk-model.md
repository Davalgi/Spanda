# Damage-Risk Model

**Status: Preview**

## Purpose

Model **harm risk**, not just errors.

## Types

`DamageRisk`, `HarmPotential`, `ProtectiveAction`, `RiskSignal`, `SafetyPainIndex`

## Examples

- Motor overheating
- Excessive vibration
- Battery swelling risk
- Operator fatigue
- Medical device abnormality
- Vehicle brake degradation

## Use

Risk-based recovery, mission abort, degraded mode, preventive maintenance, operator alert.

Entity field: `Entity.damage_risk`

See [bio-inspired-architecture.md](./bio-inspired-architecture.md).
